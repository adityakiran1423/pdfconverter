# PDF Converter

## What this script does

A script to convert a pdf into either an excel or a word file

## How to install

1. Click on the button which says Code
2. Click on download zip
    (this downloads the source code as a zip file with the file name  pdfconverter-main.zip)
3. Unzip the .zip file

## How to run

1. Type `python3 main.py <location of pdf file> <type of file you want it to convert to>`
<br> <br>
Example : `python3 main.py "Bloomberg Billionaires Index.pdf" excel`
